### Other changes shipped

- **Touch targets**: `.btn`, `.chip`, `nav.tabs button` bumped to 40–44 px min-height. Affordances mirrored to `:active` / `:focus-visible`.
- **Reduced motion**: `@media (prefers-reduced-motion: reduce)` zeroes transitions/animations.
- **System color-scheme**: `state.prefs.themeManual` distinguishes user-chosen from system-following. CSS gates dark mode on `html[data-theme-manual="0"]` so the OS preference applies until the user explicitly toggles. Migration preserves existing dark-mode users via `loadState`. Profile panel exposes "Use System Theme" when manual is set.
- **Perf**: 3 preconnects + fonts via `<link>` not `@import`. `supabase-js` and `qrious` lazy-loaded on first Manager / About nav (~60 KB saved on first paint for Lexicon-only users).

- **PWA**: `manifest.json` (standalone, ink/paper theme, 3 icons including maskable), `sw.js` (versioned cache, network-first HTML, cache-first assets, Supabase/Plausible bypass), apple-touch-icon links, install button in Profile panel wired to `beforeinstallprompt`.
