## 2. Brainstorm decisions locked this session (§5–8)

### §5 — Game loop

- **Play model:** Unlimited. Hearts inside fights only, no energy/daily cap outside.
- **Card drops:** 1 boss = 1 card. ~33 cards total.
- **Credentials:** Zone-mastery — own all 11 cards in a zone + pass mastery exam = credential issued.
- **XP:** None. Collection IS the progression. `rlfc_xp_events` table dropped from prior schema.
- **Streak:** Days played. 7-day streak makes next clear award Holo variant; 30-day makes it Legend.
- **Library role:** Reference + Leitner SRS flashcards. Counts toward streak + feeds **Study Cred** meter that unlocks cosmetic frames for owned cards. NO hints, NO tokens, NO consumables anywhere in v1.
- **Failure flow:** Streak survives (attempt = played today). Review screen shows missed Qs + correct answers. Missed terms auto-added to a "Trouble Terms" Leitner deck in Library.

### §6 — Card system

- **Catalog:** 10 + 10 + 10 standard bosses + 3 mastery exams = 33 cards.
- **Rarity earn paths (stacked):** Common (clear), Rare (perfect run), Holo (7-day streak clear), Legend (30-day streak clear OR mastery exam).
- **6 composite displays:** Common / Rare / Holo / Rare Holo / Legend / Legendary Master. Derived at render from `(perfect_ever, holo_ever, legend_ever)`. Card row never stores the composite.
- **Art:** Single illustration per card + CSS frame/foil overlays per tier (matte → redline-accent → animated foil → gold+ember glow). 33 illustrations via nano-banana pipeline.
- **Mastery exam:** 5 rounds, 90% pass, sampled across whole zone. No assistance of any kind.

### §7 — UI surfaces

- **Device target:** Mobile-first. PWA, no native.
- **Nav:** 4 bottom tabs (Map / Collection / Library / Profile). `/boss/:id` and `/credential/:zone` are full-screen takeovers without tab bar.
- **Map (home):** Zone switcher + vertical winding 11-node path per zone (Duolingo-style). All 10 standard bosses unlocked at start; mastery exam locked until 10/10.
- **Collection:** Same zone switcher; 3-wide grid of 11 cards per zone. Unearned = silhouette + greyed term name. Tap = full-screen detail with "View boss" deep link.
- **Boss-fight:** Full-screen. 3 sequential rounds, top progress bar, end card with reward reveal.
- **Welcome:** 2 fields — display_name + role (6 options); role stored + fires Plausible `signup_role` property, doesn't branch UX in v1.
- **Auth:** `/login` magic-link single field → `/auth/callback` → `/welcome` (first) or `/map` (returning). `<RequireAuth>` guards all in-game routes.

### §8 — Migration & rollout

- **Launch posture:** Hard cutover. On launch day, current static lexicon 301-redirects to `play.justennewton.media`.
- **Content migration:** Direct port of 223 terms + companion datasets (CATS, SCENARIOS, COMPAT, PLAIN_ENGLISH, REPORT_PAIRS, HAIL_THRESHOLDS, TEST_CUT_STEPS, CURRICULA) into `src/content/library.ts` as typed TS — no behavior change. 33 boss question banks (≈ 99 round files) hand-authored from term pool, NOT auto-generated.
- **Plausible goals on day 1:** `signup_completed`, `card_earned`, `credential_issued`. Engagement + retention events deferred to v2.

### Schema (final)

3 tables, all `rlfc_` prefix, RLS on `user_id = auth.uid()`:

- `rlfc_profiles` (1:1 with auth.users) — display_name, role, streak_days, last_played_at (DATE in UTC, computed from player local), study_cred, mastery_passed_zones[]
- `rlfc_earned_cards` — perfect_ever, holo_ever, legend_ever, first_earned_at, last_earned_at; PK (user_id, card_id)
- `rlfc_attempts` — score, passed, was_perfect, streak_days_at_attempt, missed_term_keys[]; immutable (no UPDATE/DELETE)

Card catalog (33 definitions) lives in TS at `src/content/cards.ts`, NOT in DB.
