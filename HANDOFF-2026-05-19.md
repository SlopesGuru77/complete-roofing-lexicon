- **Boss-fight format:** 3-round gauntlet, ~5–7 min. R1 rapid term ID, R2 scenario decision, R3 report-language gauntlet. Pass = aggregate ≥ 85%. Hearts allowed. Failure resets the fight, not progress.
- **Card rarity:** Common → Rare → Holo → Legend. ~25–40 cards total in v1.
- **Out of v1 scope:** Manager/company-seat licensing **only**. Payments, social/leaderboards, mobile native — not explicitly excluded; will be planned as "designed-for, optional, or low-effort PWA" in the remaining brainstorm sections.

- **Stack:** Vite 5 + React 18 + TypeScript + Tailwind v4 + React Router 6 + React Context+reducer + Framer Motion + Vitest. supabase-js v2. Static deploy on Zeabur. Plausible analytics (new `data-domain`). PWA manifest, no service-worker game sync v1.
- **Data model:** 4 new tables, `rlfc_` prefix — `rlfc_profiles` (1:1 with auth.users), `rlfc_earned_cards`, `rlfc_attempts`, `rlfc_xp_events`. RLS = each user sees only their own rows. - **Auth flow:** Single-screen magic-link signup/login → `/auth/callback` → first-time `/welcome` to collect display_name + role → `/map`. `<RequireAuth>` wrapper guards all in-game routes.
